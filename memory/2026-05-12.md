### MiniMax API Auth Pattern (Important — for future direct Python scripts)
- `https://api.minimax.io/anthropic/v1/messages` expects `X-Api-Key` header
- But the API key from `openclaw.json` is OpenClaw-gateway-proxied — direct urllib calls fail with 401
- Working pattern: use OpenClaw's `sessions_spawn` with MiniMax model, or use the OpenClaw gateway proxy at `127.0.0.1:18789`
- `trendscout_gen.py` and `synthesize_checklist.py` work because they read credentials from `openclaw.json` and use `Authorization: Bearer` header — works through OpenClaw's proxy
- For standalone scripts outside OpenClaw: embed via `sessions_spawn` instead of raw HTTP
