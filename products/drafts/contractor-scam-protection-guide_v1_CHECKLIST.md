# CONTRACTOR SCAM PROTECTION CHECKLIST

## 15-Minute Vetting Protocol for Homeowners

---

## BEFORE YOU SIGN

**License + Identity**
- [ ] License number verified on state CSLB website
- [ ] Name on license matches name on contract exactly
- [ ] License is active (not expired or suspended)
- [ ] Insurance COI received within 24 hours ($1M liability minimum)
- [ ] Business registered with Secretary of State

**Contract Review**
- [ ] Start and completion dates included
- [ ] Detailed scope of work and materials list
- [ ] Payment milestones tied to completion
- [ ] Down payment ≤ 33% of total contract value
- [ ] Warranty clause included
- [ ] Lien waiver clause included
- [ ] No "assignment of benefits" document requested
- [ ] No blank change orders to sign

**Payment Rules**
- [ ] No more than 1/3 upfront
- [ ] Never pay in cash
- [ ] Check, credit card, or protected payment platform only

---

## RED FLAGS — WALK AWAY IF:
- License doesn't verify on state board
- More than 33% required upfront
- Cash only
- No written contract
- Door-to-door solicitor
- "Today's the last day for this price"
- Business registered < 1 year ago with no track record
- Can't produce Certificate of Insurance within 24 hours

---

## IF YOU'VE ALREADY BEEN SCAMMED

1. [ ] Document everything (contract, receipts, photos, all communications)
2. [ ] File a complaint with your state Contractors State License Board
3. [ ] File a police report (amount above state theft threshold — typically $950+)
4. [ ] File an FTC complaint at reportfraud.ftc.gov
5. [ ] Contact your insurance company if property damage occurred
6. [ ] Consult a construction law attorney for mechanics lien + civil options
7. [ ] Small claims court if amount is within limits

**Act fast. Evidence disappears.**
