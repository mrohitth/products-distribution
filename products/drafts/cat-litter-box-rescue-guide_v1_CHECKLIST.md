# 7-Day Litter Box Reset Checklist

**Companion to the Cat Litter Box Rescue Guide**

---

## Before You Start

- [ ] Vet appointment booked (urinalysis + physical exam)
- [ ] Rule out UTI / FLUTD / kidney disease before proceeding

---

## Day 1: Box Audit

- [ ] Count boxes: N cats + 1 = minimum boxes
- [ ] Check size: 1.5x cat length, no hoods/lids
- [ ] Check location: quiet, no dead-end corners, two escape routes
- [ ] Remove litter liners (cats hate the crinkle)

## Day 2: Litter Swap

- [ ] Unscented, clumping, fine-grained (Dr. Elsey's preferred)
- [ ] Fill 3-4 inches deep, not shallow
- [ ] Keep ONE box with old litter as transition option

## Day 3: Deep Clean

- [ ] Dump all litter, wash boxes with hot water + mild dish soap
- [ ] NO ammonia cleaners, NO bleach, NO vinegar
- [ ] Dry thoroughly before refilling
- [ ] Apply enzyme cleaner to all accident spots (Rocco & Roxie or Anti-Icky Poo)

## Day 4: Stress Assessment

- [ ] Check for multicat tension (hissing, blocking, staring)
- [ ] Add Feliway diffuser in problem area
- [ ] Create vertical space (shelves, cat trees, window perches)
- [ ] Separate food/water from litter boxes

## Day 5: Scoop Protocol

- [ ] Scoop minimum 2x daily
- [ ] Full dump + wash every 2 weeks
- [ ] Line boxes with newspaper only (no plastic liners)

## Day 6: Observation

- [ ] Track elimination: box vs outside, frequency, consistency
- [ ] Note any straining, blood, or crying during urination
- [ ] Flag anything concerning → return to vet

## Day 7: Reset Complete

- [ ] Review tracking log for patterns
- [ ] One change at a time going forward (not everything at once)
- [ ] If resolved: maintain the system
- [ ] If unresolved: consult a veterinary behaviorist
