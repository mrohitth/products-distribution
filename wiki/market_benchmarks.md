## Cat Care Notion Templates — Market Benchmarks

### Strategic Context
The "Pet Health / Cat Care" Notion template niche is **moderately competitive** with established sellers on Gumroad, Etsy, and Notionery. Most templates span $7–$19. Top-selling features include vaccination tracking, vet visit logs, and grooming schedules.

| Rank | Title | Price | Key Bonuses | Notes |
|------|-------|-------|-------------|-------|
| 1 | **Pet Health & Wellness Tracker** (Notionery/Templateista) | $12–$15 | Vaccination calendar, medication log, weight tracker, food diary, vet visit notes | Consistently top-rated in "pet" category |
| 2 | **Cat Care Bible** (Etsy — various sellers) | $9–$14 | Feeding schedule, litter box tracker, health record, grooming log, printable PDF version | High volume on Etsy search |
| 3 | **Ultimate Pet Owner Dashboard** (Gumroad — template bundles) | $15–$19 | Multi-pet support, expense tracking, appointment scheduler, emergency contact database | Premium tier with most features |

### Price Band Observations
- **Entry ($5–$10):** Minimalist single-page trackers (health only) — low differentiation.
- **Mid ($10–$18):** Most competitive — calendars + logs + checklists + some automation.
- **Premium ($18–$30):** Multi-pet, expense tracking, grooming schedules, connected databases.
- **Top sellers** tend to be $12–$16 with clear previews and "lifetime updates" as a bonus.

| Feature | Frequency | Buyer Signal |

| Printable version | Medium | Etsy buyers love PDFs |

## Appliance DIY / Home Repair Notion Templates — Market Benchmarks

### Strategic Context
The "Appliance Repair / Home Maintenance" Notion niche is **lower competition** than pet care but has a smaller total addressable market. Most templates target general home maintenance rather than specific appliance brands.

| Rank | Title | Price | Key Bonuses | Notes |
|------|-------|-------|-------------|-------|
| 1 | **Home Maintenance Hub** (Notionery / Gumroad) | $12–$16 | Annual maintenance calendar, appliance age tracker, warranty document storage, repair log | Top seller in "home" category |
| 2 | **DIY Home Repair Tracker** (multiple Etsy sellers) | $8–$14 | Error code reference template, parts purchase tracker, project log, contractor contacts | Mid-tier with error code feature |
| 3 | **Home Appliance Manager** (Gumroad — niche sellers) | $10–$15 | Brand-specific error code database, maintenance intervals, repair history, warranty expiration tracker | Most specific to appliances — less generic home maintenance |

### Price Band Observations
- **Entry ($5–$10):** Single-page repair logs — basically a fancy checklist.
- **Mid ($10–$16):** Multiple linked databases (appliances → repairs → parts → contractors).
- **Premium ($16–$25):** Error code databases, QR code document storage links, integration with calendar.
- **Differentiator opportunity:** Appliance-specific error code databases (Bosch E24, Whirlpool F21, etc.) are NOT commonly offered — this is a gap.

## Key Market Gaps Identified

### 🟢 Greenfield Opportunities

1. **Brand-specific error code databases** — Nobody is doing Bosch-specific E24/E15 guides as Notion templates. This is a clear gap.
2. **Cat breed-specific care templates** — General "pet health" is saturated. A Ragdoll-specific template (with HCM screening reminders, grooming frequency for semi-longhair, bladder stone prevention) would stand out.
3. **Combined "Home Appliance X-Ref"** — Linking appliance model numbers → common error codes → video fix links → parts distributor links creates a dependency graph most templates don't offer.

### 🔴 Saturated Zones to Avoid
- Generic single-page pet health trackers ($5–$10 range).
- Basic home maintenance calendars without error code integration.
